# Day 11 - Topic 7: NAP Standardization (Name, Address, Phone)

## Overview

NAP standardization - ensuring your Name, Address, and Phone number are identical across every platform - is one of the fastest ways to boost your AI discoverability. When AI models scan the internet and find conflicting information about you, they filter you to the bottom of recommendations as potentially unreliable. Perfect consistency signals credibility and authority.

This seemingly simple task has massive impact: agents who standardize their NAP across all platforms typically see 30-50% improvement in AI recommendation rates within 30-60 days.

## Common NAP Inconsistency Issues

### Issue 1: Phone Number Formatting

**The Problem:**
```
Google: (201) 555-1234
Facebook: 201-555-1234
Zillow: 201.555.1234
LinkedIn: +1 201 555 1234
```

**The Solution:**
Pick ONE format and use everywhere:
```
Recommended: (201) 555-1234
Acceptable: 201-555-1234
```

**Never Mix:** Using both formats across platforms confuses AI

---

### Issue 2: Name Variations

**The Problem:**
```
Google: John Smith - Realtor
Facebook: John Smith Real Estate
LinkedIn: John A. Smith, Realtor
Zillow: J. Smith - Agent
```

**The Solution:**
```
Standard Format: John Smith - Realtor
Use Everywhere: Exact same format
No Variations: No middle initials, no abbreviations
```

**Exception:** LinkedIn allows longer headline, but keep name portion identical

---

### Issue 3: Address Format Inconsistencies

**The Problem:**
```
Platform 1: 123 Main St, Mahwah, NJ 07430
Platform 2: 123 Main Street, Suite 200, Mahwah, NJ 07430
Platform 3: 123 Main Street, Mahwah, New Jersey
Platform 4: 123 Main St., Mahwah NJ
```

**The Solution:**
```
Standard: 123 Main Street, Suite 200, Mahwah, NJ 07430

Include:
âœ… Full "Street" not "St"
âœ… Suite/Unit number
âœ… City name
âœ… State abbreviation (NJ not New Jersey)
âœ… ZIP code
âœ… Comma after city
```

---

### Issue 4: Multiple Email Addresses

**The Problem:**
```
Google: john.smith@christiesreg.com
Facebook: johnsmith@gmail.com
Zillow: jsmith@christiesreg.com
Personal site: john@johnsmithhomes.com
```

**The Solution:**
- Pick ONE primary email
- Use on ALL platforms
- Forward other emails to primary
- Update old listings

**Recommended:** Use brokerage email as primary (looks more professional)

---

### Issue 5: Brokerage Name Variations

**The Problem:**
```
Platform 1: Christie's
Platform 2: Christie's International Real Estate
Platform 3: Christie's Real Estate Group
Platform 4: Christie's International Real Estate Group
```

**The Solution:**
```
Legal Name: Christie's International Real Estate Group
Use Everywhere: Exact same spelling
Check Your License: Use what's on your license
```

## Maintenance & Ongoing Management

### Quarterly NAP Audit

**Every 3 Months:**

1. **Review Master Template**
   - Is all information current?
   - Has phone changed?
   - Has email changed?
   - Has address changed?

2. **Spot Check 5 Platforms**
   - Google Business
   - Facebook
   - LinkedIn
   - Zillow
   - Personal website

3. **Update Template if Needed**
   - Revise master template
   - Note update date
   - Plan platform updates

4. **Update All Platforms**
   - If anything changed
   - Use same systematic process
   - Document in audit spreadsheet

---

### When You Change Offices

**If You Move to New Office:**

1. **Update Master Template First**
   - New address
   - New office phone (if applicable)
   - Same personal phone (recommended)

2. **Update Google Business IMMEDIATELY**
   - Triggers re-verification
   - Critical for local search

3. **Update All Other Platforms (Same Day)**
   - Tier 1 platforms
   - Then Tier 2-4

4. **Why Speed Matters:**
   - Old address creates inconsistency
   - AI detects conflict quickly
   - Rankings can drop while inconsistent

---

### When You Get New Phone Number

**Process:**

1. **Update Master Template**
2. **Keep Old Number Active (Temporary)**
   - Forward to new number for 30 days
   - Prevents missed calls during transition
3. **Update All Platforms Systematically**
   - Start with Tier 1
   - Complete all within 1 week
4. **Update Print Materials**
   - Business cards
   - Signs
   - Marketing materials
