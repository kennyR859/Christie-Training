## The Rate vs Fees Trade-Off

### Understanding APR vs Interest Rate

**Interest Rate:**
- The actual percentage charged on loan principal
- What you see advertised
- Determines monthly payment

**APR (Annual Percentage Rate):**
- Includes interest rate PLUS all fees
- Shows true cost of loan
- Better comparison metric
- **ALWAYS compare APR, not just interest rate**

**Example:**
```
Lender A:
Interest Rate: 6.5%
Fees: $500
APR: 6.55%

Lender B:
Interest Rate: 6.25%
Fees: $8,500
APR: 6.75%

Lender A is CHEAPER despite higher interest rate!
```

### Points and Rate Buydowns

**What Is a Point?**
- 1 point = 1% of loan amount
- Used to "buy down" interest rate
- Permanent rate reduction

**When Points Make Sense:**
```
$500,000 Loan:

Option 1: 7% rate, 0 points
Monthly payment: $3,327
Total paid Year 1: $39,924

Option 2: 6.5% rate, 1 point ($5,000)
Monthly payment: $3,160
Total paid Year 1: $42,920 ($37,920 + $5,000 point)

Break-even: Month 30 (2.5 years)

If staying 3+ years: Buy the point
If selling in 2 years: Don't buy the point
```

**Pat's Guidance:**
> "Points can save money long-term, but only if you're keeping the loan 3+ years. If you might refinance or sell soon, points are wasted money."

## How to Compare Lender Estimates

### The Loan Estimate Form

**Federal Law (TRID):**
- All lenders MUST use same Loan Estimate form
- Issued within 3 days of application
- Shows all costs in standardized format
- Easy comparison between lenders

**Key Sections to Compare:**

**Section A - Origination Charges:**
```
Lender A: $2,500 (fees + points)
Lender B: $0
Lender C: $4,800

Unity: Typically $0-1,000 (points only if portfolio/construction)
```

**Section B - Services You Cannot Shop For:**
```
Appraisal, credit report, flood cert
Should be similar across lenders ($600-800 total)
```

**Section C - Services You Can Shop For:**
```
Title insurance, survey, pest inspection
Can choose your own providers
Unity uses Regency Title (agent benefits)
```

**Section E - Taxes and Government Fees:**
```
Recording fees, transfer taxes
Same regardless of lender
```

**Section F - Prepaids:**
```
Property taxes, insurance, interest
Same regardless of lender
```

**Section G - Initial Escrow:**
```
Reserves for taxes and insurance
Same regardless of lender
```

**Total Cash to Close:**
Add all sections = what buyer needs at closing
